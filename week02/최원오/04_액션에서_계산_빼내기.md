## 공부 내용

- 순수함수를 위해서는 전역 변수가 없어야하며 함수가 결괏값을 리턴해야한다.
- 함수는 입력과 출력이 있다.
- 명시적 입출력 - 부수효과가 없음
- 암묵적 입출력 - 부수효과 발생

### 액션에서 계산 추출하기

1. 계산 코드를 분리하기
2. 계산 함수에 암묵적 입력과 출력 찾기
3. 암묵적 입력은 인자로 암묵적 출력은 리턴 값으로 변경하기

- 79p 연습문제

```
const caculateTax = (totalPrice) => {
return totalPrice * 0.10
}
```

const checkExceedFreeDelivery = (price, total) => {
